# Calculadora de interés simple

Landing con tres secciones (la primera implementada) para practicar cálculos de interés simple. Incluye validación de entradas, selección de incógnita y un botón de evaluación que compara el resultado del usuario con el cálculo esperado.

## Uso
1. Abre `index.html` en tu navegador.
2. Completa los valores conocidos, selecciona la incógnita (C, i, n o I), escribe la fórmula que planeas usar e ingresa tu resultado.
3. Pulsa **Evaluar respuesta** para verificar tu cálculo; usa **Limpiar** para reiniciar el formulario.

Los valores deben ser numéricos, positivos y diferentes de cero. El cálculo tolera un margen de ±0,5 % para validar la respuesta.
