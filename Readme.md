# Calculadoras de interés

Landing con tres secciones para practicar cálculos de interés simple, compuesto y simulación de créditos con tabla francesa o alemana. Incluye validación de entradas, selección de incógnita y botones de acción para validar o calcular resultados.

## Uso
1. Abre `index.html` en tu navegador.
2. Sección 1 y 2: completa los valores conocidos, selecciona la incógnita, escribe la fórmula que planeas usar e ingresa tu resultado.
3. Pulsa **Evaluar respuesta** para verificar tu cálculo; usa **Limpiar** para reiniciar el formulario.
4. Sección 3: ingresa monto, tasa anual, años, pagos por año y el tipo de tabla (francesa o alemana), escribe la fórmula de referencia y pulsa **Calcular tabla** para ver todas las cuotas.

Los valores deben ser numéricos, positivos y diferentes de cero. Las validaciones impiden dejar vacías las incógnitas y la tasa se considera nominal anual (dividida según la frecuencia). Para las secciones 1 y 2 se tolera un margen de ±0,5 % al validar la respuesta del usuario.
